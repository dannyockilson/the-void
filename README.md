# The Void

A cathartic web app about screaming into the void.

**Live Site:** https://dannyockilson.github.io/the-void/

## About

The Void Screamer is a dark, immersive experience that captures your screams through the Web Audio API and transforms them into visual feedback on a canvas. Release your frustrations into the digital abyss.

## Development

No build setup required for the MVP - just open `index.html` in a browser or serve it locally:

```bash
python -m http.server
```
